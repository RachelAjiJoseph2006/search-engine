# Deadass RAG Search 🔍

A simple **Retrieval-Augmented Generation (RAG)** web app that lets users **upload a PDF** and **ask questions** about its content.

The app extracts text from the PDF, stores embeddings, retrieves relevant chunks, and generates answers using an LLM — all through a clean Google-style UI.

---

## ✨ Features

- 📄 Upload a PDF
- 🔎 Ask natural language questions
- 🧠 RAG-based answers (context-aware)
- 📚 Source chunks shown with answers
- 🎨 Clean Google-inspired frontend UI
